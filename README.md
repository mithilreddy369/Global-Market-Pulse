# Key Features

1. High-Focus UI: An ultra-dark (navy & black) theme with high-contrast gold accents, built for real-time monitoring with minimal eye strain.

2. Multi-Asset Tracking: Consolidated view of global indices, key Indian market sectors, major Forex pairs, commodities, and cryptocurrencies.

3. Self-Contained & Fast: A single, lightweight HTML/JS file that automatically refreshes data every 60 seconds using public APIs.
